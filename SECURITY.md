# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in envrizz, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please report them via GitHub's private vulnerability reporting:

1. Go to [https://github.com/terrancemacgregor/envrizz/security/advisories/new](https://github.com/terrancemacgregor/envrizz/security/advisories/new)
2. Fill out the vulnerability details
3. Submit the report

You can expect an initial response within 48 hours. We will work with you to understand and address the issue before any public disclosure.
